# LxL 3D Printing Website

Professional website for LxL 3D Printing business with QR code and printable table tent.

## Quick Setup for GitHub Pages

1. Initialize git (if not already done):
   ```bash
   git init
   git add .
   git commit -m "Initial commit: LxL 3D Printing website"
   ```

2. Connect to your GitHub repository:
   ```bash
   git remote add origin https://github.com/jtwm-0677/LxL3D-Printing.git
   git branch -M main
   git push -u origin main
   ```

3. Enable GitHub Pages:
   - Go to your repository: https://github.com/jtwm-0677/LxL3D-Printing
   - Click "Settings" tab
   - Click "Pages" in the left sidebar
   - Under "Source", select "main" branch
   - Click "Save"
   - Your site will be live at: https://jtwm-0677.github.io/LxL3D-Printing

## Files Included

- `index.html` - Main website with contact form
- `table-tent.html` - Printable table tent with QR code
- `README.md` - This file

## Printing the Table Tent

1. Open `table-tent.html` in your browser
2. Wait for the QR code to generate (appears automatically)
3. Press Ctrl+P (or Cmd+P on Mac) to print
4. Select "Print to PDF" or print directly
5. Fold the paper in half along the dotted line (portrait orientation)
6. Both sides will face outward when folded

## QR Code

The QR code is automatically generated and points to:
`https://jtwm-0677.github.io/LxL3D-Printing`

The QR code will appear on the table tent once you open it in a browser.

## Customization

To customize the website:

- Update the tagline or business description
